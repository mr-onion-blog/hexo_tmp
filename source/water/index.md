## 生活饮用水卫生标准

**标准号：GB 5749-2022**

_实施日期：2023年4月1日_

适用范围：市政供水、自备水源、管道直饮水等直接饮用的水源

| /              | 数据                                                      |
| -------------- | --------------------------------------------------------- |
| 微生物指标     | 大肠菌群、耐热大肠菌群不检出                              |
| 毒理指标       | 重金属（铅 <= 0.01mg/L ）、农药残留（百菌清 <= 0.01mg/L） |
| 感官性状指标   | 浑浊度 <= 1NTU，无臭无异味                                |
| 消毒剂及副产物 | 管网末梢游离氯 >= 0.3mg/L                                 |
